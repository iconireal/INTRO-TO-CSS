The <marquee> Tag

The <marquee> tag is an HTML element used to create a scrolling area of text or images.

Attributes

Layout Attributes

Attribute Description Example
width Specifies the width of the marquee 10, 20%, etc.
height Specifies the height of the marquee 10, 20%, etc.

Scrolling Attributes

Attribute Description Example
direction Specifies the direction of scrolling up, down, left, right
behavior Specifies the type of scrolling scroll, slide, alternate
scrolldelay Specifies the delay between each jump 10, etc.
scrollamount Specifies the speed of the marquee text 10, etc.

Loop and Appearance Attributes

Attribute Description Example
loop Specifies the number of times to loop INFINITE, etc.
bgcolor Specifies the background color color name or hex value
hspace Specifies the horizontal space around the marquee 10, 20%, etc.
vspace Specifies the vertical space around the marquee 10, 20%, etc.

  • History of HTML:

    1. Early Days (1980s): Tim Berners-Lee, a British computer scientist, invented HTML in the late 1980s while working at CERN. He designed HTML to facilitate the sharing of scientific documents over the internet. 2. First Version (1993): The first version of HTML, HTML 1.0, was published in 1993. It was a simple markup language with limited features. 3. HTML 2.0 (1995): HTML 2.0 was published in 1995, introducing features like tables, forms, and client-side image maps. 4. HTML 3.2 (1997): HTML 3.2 was published in 1997, adding features like tables, applets, and style sheets. 5. HTML 4.01 (1999): HTML 4.01 was published in 1999, introducing features like improved table support, accessibility features, and internationalization. 6. XHTML (2000s): XHTML (Extensible HyperText Markup Language) was introduced in the early 2000s, aiming to make HTML more extensible and compatible with XML. 7. HTML5 (2014): HTML5 was published in 2014, revolutionizing web development with features like: - Improved multimedia support (video, audio, canvas) - Enhanced semantic elements (header, footer, nav, section) - Offline storage and caching - Improved accessibility features 8. Current Status (2023): HTML continues to evolve, with ongoing work on new features, accessibility improvements, and better support for modern web development needs.

  • Today's HTML (Our amazing package)
    HTML's history and ongoing development have transformed the web into the powerful, interactive, and accessible medium we know today
    • HTML remains the foundation of the web, and its evolution continues to shape the way we build and interact with web applications. With the rise of modern web development frameworks and libraries, HTML's role has expanded to include:
    • Semantic HTML: Using HTML elements to convey meaning and structure, rather than just presentation.
    • Accessibility: Ensuring web pages are accessible to users with disabilities, following guidelines like the Web Content Accessibility Guidelines (WCAG).
    • Responsive Design: Creating web pages that adapt to different screen sizes, devices, and orientations.
  • What is HTML?
    HTML (HyperText Markup Language) is a standard markup language used to create web pages. It provides the structure and content of a webpage, allowing web browsers to interpret and display the page's layout, text, images, and other multimedia elements.
  • Basic structure of an HTML document:
    <!DOCTYPE html>
<html>
   <head>
     <title>Page Title</title>
   </head>
   <body>
     <!-- Content goes here -->
   </body>
</html>
    • DOCTYPE declaration: Declares the document type and version of HTML.
    • html element:The root element of the HTML document.
    • head element:Contains metadata about the document, such as the title, charset, and links to external stylesheets or scripts.
    • title element:Sets the title of the page, which appears in the browser's title bar and in search engine results
    • body element:Contains every other content of the HTML document.
  • HTML Elements and Tags:
    • Block-level elements: <div>, <p>, <h1>-<h6>, <ul>/<li>
    • Inline elements: <span>, <a>, <img>, <strong>
  • Attributes and Values:
    • Common attributes: id, class, style, href, src
    • Boolean attributes: checked, disabled
